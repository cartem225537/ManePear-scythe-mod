# ManePear's Scythe Mod (Fabric 1.21.11)

Коса с механикой палицы, отключением щита и кастомным зачарованием Жатва (Reaping).

## Как собрать .jar с телефона (через GitHub Actions)

У тебя нет ПК — собираем на серверах GitHub. Это бесплатно.

1. Зайди на github.com, создай аккаунт (если нет) — можно с телефона.
2. Нажми "+" вверху справа -> **New repository**. Назови как хочешь, тип **Public**, создай.
3. На странице репозитория: **Add file** -> **Upload files**.
4. Распакуй присланный zip и загрузи ВСЁ его содержимое (включая папки `.github`, `src`, `gradle`, файлы `build.gradle`, `gradlew` и т.д.). На телефоне проще через приложение GitHub или файловый менеджер с поддержкой zip.
5. Нажми **Commit changes**.
6. После загрузки сборка запустится сама. Открой вкладку **Actions** вверху.
7. Дождись зелёной галочки (3-5 минут). Если красный крест — открой запуск, скопируй текст ошибки и пришли мне, я поправлю.
8. Внутри успешного запуска внизу будет блок **Artifacts** -> **manepears-scythe-jar**. Скачай его. Внутри zip лежит `.jar`.
9. Из `build/libs/` нужен файл `manepears-scythe-1.0.0.jar` (НЕ `-sources.jar`).

Готовый `.jar` кидаешь в папку `mods` рядом с Fabric API и Fabric Loader 0.18.4+.

## Если версия yarn не та

Если сборка падает на mappings — открой `gradle.properties`, поправь строку
`yarn_mappings=1.21.11+build.1` на актуальный build (посмотри на https://fabricmc.net/develop)
и снова закоммить.

## Что делает мод

- Коса: 10 урона, скорость атаки 1.6, smash-атака и отключение щита как у палицы.
- Поддержка зачарований палицы и меча.
- Жатва (1-3): крадёт по 0.5 сердца за уровень при ударе по игроку, лечит владельца.
- Книга Жатвы только в сундуках Ancient City (1% ур.1-2, 0.5% ур.3), не у торговцев.
- Крафт 3x3: N D N / T M T / T S T.
- Достижения Mower и Vampire во вкладке Adventure.
